The Interface

The editor is divided into three main areas:

1

Sidebar (Left)

Manage files, uploaded images, and custom fonts. Toggle with Ctrl+B.

2

Editor (Center)

Write your Typst code here. Features syntax highlighting and autocomplete.

3

Preview (Right)

See your compiled PDF in real-time. Zoom and navigate pages.

üí° Tip

Press Ctrl+Space anywhere in the editor to trigger autocomplete suggestions!

Text Formatting

Typst offers several ways to format your text:

Syntax Result Description
*bold* bold Bold text
_italic_ italic Italic text
*_both_* both Bold and italic
`code` code Inline code
#underline[text] text Underlined
#strike[text] text Strikethrough

Advanced Text Styling

Typst 
// Change text size
#text(size: 18pt)[Larger text]

// Change text color
#text(fill: blue)[Blue text]
#text(fill: rgb("#ff5733"))[Custom color]

// Combined styling
#text(size: 14pt, fill: red, weight: "bold")[Styled text]

Typst

Images

To use images in your document:

1

Upload the Image

Drag and drop your image onto the Uploads section in the sidebar, or click the upload button.

2

Reference in Code

Use the #image() function with your filename.

Typst 
// Basic image
#image("photo.png")

// With width constraint
#image("photo.png", width: 50%)

// Image with caption (figure)
#figure(
  image("photo.png", width: 80%),
  caption: [A beautiful photograph]
)
üìå Note

Supported formats: PNG, JPG, JPEG, GIF, SVG, WebP

Tables

Create tables with the #table() function:

Typst 
// Simple table
#table(
  columns: 3,
  [*Name*], [*Age*], [*City*],
  [Alice], [28], [New York],
  [Bob], [32], [London],
  [Carol], [25], [Tokyo],
)

Styled Table

Typst 
#table(
  columns: (1fr, 2fr, 1fr),
  align: (left, center, right),
  stroke: 0.5pt + gray,
  inset: 10pt,
  
  // Header with background
  table.header(
    table.cell(fill: blue.lighten(80%))[*Item*],
    table.cell(fill: blue.lighten(80%))[*Description*],
    table.cell(fill: blue.lighten(80%))[*Price*],
  ),
  
  [Apple], [Fresh red apple], [$1.50],
  [Banana], [Organic banana], [$0.75],
)

Page Setup

Configure your document's appearance with #set rules:

Typst 
// Page configuration
#set page(
  paper: "a4",
  margin: (x: 2.5cm, y: 3cm),
)

// Text settings
#set text(
  font: "New Computer Modern",
  size: 11pt,
)

// Paragraph settings
#set par(
  justify: true,
  leading: 0.65em,
)

// Heading numbering
#set heading(numbering: "1.1")

Headers & Footers

Typst 
#set page(
  header: [
    _My Document_
    #h(1fr)
    #datetime.today().display()
  ],
  footer: [
    #h(1fr)
    Page #counter(page).display()
    #h(1fr)
  ],
)

Custom Fonts

Upload and use your own fonts:

1

Upload Font File

Click the Fonts section and upload your TTF, OTF, WOFF, or WOFF2 file.

2

Use the Font

Reference it by name in your document:

Typst 
// Set for entire document
#set text(font: "My Custom Font")

// Or use for specific text
#text(font: "My Custom Font")[This uses the custom font]
üí° Tip

When typing font: ", autocomplete will show your uploaded fonts first!

Tips & Tricks

1. Use Functions for Reusable Styles

Typst 
// Define a reusable note box
#let note(body) = box(
  fill: yellow.lighten(80%),
  inset: 10pt,
  radius: 4pt,
  body
)

// Use it anywhere
#note[This is an important note!]
#note[Another note here.]

2. Quick Comments

Typst 
// This is a single-line comment

/* This is a
   multi-line
   comment */

3. Links & References
